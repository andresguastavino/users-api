Welcome to my Users API :)

  • Consult the Users API endpoint (all users) at /api/all
  • Consult the Users API endpoint with unvalidated pagination (i.e. if you fetch page=-1 it will let you but will not return any results) at /api/users?page={pageId}
  • Consult the Users API endpoint with validated pagination (i.e. if you fetch page=-1 it will return page=0, or if you fetch page=6 and count (total number of users) is 43 it will return page=4) appending "&validated" to the unvalidated pagination endpoint /api/users?page={pageId}&validated
  • Add "pretty" query to the endpoint if you want to prettify it :) i.e. /api/all?pretty
